# HELIOS ONLINE

HELIOS ONLINE is a browser-based multiplayer solar system exploration game built with Vite and Three.js.

## Stack

- Vite
- Three.js
- cannon-es
- socket.io-client

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Current Scope

- Phase 1: solar system engine, orbital mechanics, celestial bodies, belts, rings, UI
- Phase 2: spaceship controls, geolocation spawn, cameras, HUD
- Phase 3: planetary surfaces, landing environments, hazards, and transition system

## Notes

- Best experience is on a desktop browser with WebGL enabled.
- Geolocation features require browser permission.
